Jhapiv1

Android Library platform for the Jotihunt

View on GitHub Download .zip Download .tar.gz

Introduction

Jhapiv1 stands for JotiHunt API V1, it is a library platform consisting out of 3 libraries. The Core library(required for the other two), is the foundation of the Jhapiv1, it provides enumeration and networking. The Maps Library that provides support for the Google Maps Android API, and the UI library which provides several fragments & views for displaying jotihunt data(such as the news).

Core

The foundation of the Android Jotihunt API, it provides models for the API and the base enumeration.

Installation

compile 'nl.jhcdo.jotihunt:jotihunt-core:1.0'

Maps

Provides the Deelgebied Api and various map/location utils, it requires the core.

Installation

compile 'nl.jhcdo.jotihunt:jotihunt-maps:1.0'

UI

Provides views and fragments for displaying data, it requires the core.

Installation

compile 'nl.jhcdo.jotihunt:jotihunt-ui:1.0'
